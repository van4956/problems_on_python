# problems_on_python

Добрый день!

В этом репозитории находится подборка интересных задач на Python.  
Задачи различных онлайн-курсов, онлайн-тренингов, митапов, олимпиад, а также задачи с LeetCode.  
Все задачи реализованы в Jupyter Notebook.
